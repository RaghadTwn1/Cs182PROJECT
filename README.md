# 🚘 Luxury Car Rental System

AI-inspired luxury car rental management system built using C++ and Object-Oriented Programming concepts.

---

# ✨ Project Overview

This project simulates a real luxury car rental agency system.

The system manages:
- 🚗 Luxury Cars
- 👤 Clients
- 👨‍💼 Employees
- 💳 Rental Operations

It also demonstrates important OOP concepts including inheritance, multiple inheritance, and operator overloading.

---

# 🌟 Features

### 🚗 Available Cars Management
Displays all available luxury cars in a clean organized format.

### ⚖️ Car Comparison
Compares cars using:
- == operator
- < operator

### 💳 Rental Processing
Processes a 5-day rental simulation and calculates the total cost.

### 🔄 Availability Update
Automatically changes car status after rental.

### 👨‍💼 Employee Information
Displays employee details using multiple inheritance.

# 📂 Classes Included

### 👤 Person
Base class for:
- Client
- Employee

### 💎 Client
Stores client information and ID.

### 💼 Worker
Stores:
- Position
- Salary

### 👨‍💼 Employee
Derived from:
- Person
- Worker

### 🚘 Car
Stores:
- Brand
- Model
- Year
- Daily Price
- Availability

### 🏢 Agency
Main system controller responsible for:
- Managing cars
- Managing clients
- Managing employees
- Processing rentals
